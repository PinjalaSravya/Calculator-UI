Calculator UI

This project is a simple calculator user interface created using HTML and CSS.
It focuses on layout, structure, and responsive design without implementing calculation logic.
1)Features
 Clean and simple calculator layout
 Responsive design suitable for different screen sizes
 Organized button grid and display area
 Pure HTML and CSS (no JavaScript)
2)Technologies Used
 HTML
 CSS
3)roject Purpose
 This project was built to practice:
 Web page structure using HTML
 Layout techniques using CSS
 Designing user interfaces without relying on JavaScript logic

4)How to Use
 Clone or download the repository
 Open the index.html file in any web browser

5)Limitations
 This is a UI-only project
 Does not perform arithmetic calculations
 No JavaScript functionality included

6)Future Improvements
 Add JavaScript to enable calculator operations
 Improve styling with animations or themes
Developed as part of learning and practicing front-end fundamentals.
